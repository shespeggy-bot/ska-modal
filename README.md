# SKA 弹窗静态页

面向设计的移动端弹窗预览页面（与 Figma 稿「弹窗1」节点对应）。无需安装 Node、无需打包，用浏览器即可查看。

---

## 本地运行

适合在**自己电脑**上先看效果，再交给开发接入业务。

### 方法一：直接打开文件（最快）

1. 在文件夹里**双击** `index.html` 或 `modal.html`，会用系统默认浏览器打开。  
2. 若样式或图片异常，优先用**方法二**。

### 方法二：用「本地小网站」打开（推荐）

这样地址是 `http://开头`，更接近真实线上环境。

1. 打开 **终端**（Mac：「终端」应用；或在 Cursor 里 **Terminal → New Terminal**）。  
2. 输入下面两行，每行末尾按 **回车**（路径请改成你本机里这个项目的实际位置）：

```bash
cd "/Users/jazz/Documents/CURSOR test"
python3 -m http.server 8765
```

3. 看到类似 `Serving HTTP on ... port 8765` 表示成功。  
4. 打开浏览器，地址栏输入：

   - **带预览壳的整页：**  
     `http://127.0.0.1:8765/index.html`
   - **只要弹窗全屏页：**  
     `http://127.0.0.1:8765/modal.html`

5. 看完后回到终端窗口，按 **Ctrl + C** 可停止服务。

> **说明：** 若本机没有 `python3`，可请开发同事用任意「静态文件服务器」工具代替，或直接把 `modal.html` 交给开发嵌入项目。

### 关于 Figma 切图（可选）

部分装饰图在稿里来自 **Figma 桌面端** 的本地地址（`localhost:3845`）。  
**未开 Figma 时**，页面会自动用占位或简化图形，属正常现象；上线前需由开发把正式图片放进项目并改链接。

## 目录说明

| 名称 | 说明 |
|------|------|
| **index.html** | **主预览页**：左右为占位侧栏，中间为手机宽度预览框，内含弹窗；适合在评审时展示「画布 + 弹窗」整体关系。 |
| **modal.html** | **仅弹窗页**：全屏灰底 + 弹窗内容，更接近真实 H5 里「只弹这一层」的效果；给开发对照标注时优先看这个。 |
| **.gitignore** | 给 Git 用的忽略规则（例如忽略系统产生的 `.DS_Store`），**不用改、不用打开**。 |
| **README.md** | 本说明文档。 |

当前仓库**没有**复杂子文件夹；所有样式和结构都在上述 HTML 文件里，设计师改文案或交给开发时，指明文件名即可。

## 给设计同学的一句话

- 日常看图：优先 **`modal.html`**（本地运行方法二打开）。  
- 需要给开发/测试一个可访问地址时：把项目推到 **GitHub / GitLab**，由部署平台生成链接（如 Vercel、公司内网部署）。
